# 대회 분석

분석 기준일: 2026-07-06 KST
대회명: 제4회 인공지능(AI) 신약개발 경진대회(4th JUMP AI, fourth.py) 예선
공식 페이지: https://daker.ai/public/hackathons/4th-jump-ai-agentic-drug-challenge

## 1. 한 줄 결론

이 대회는 단순 예측 모델보다, 신약개발 전주기 문제를 스스로 정의하고 도구를 호출하며 검증 루프를 수행하는 **Agentic AI 서비스 설계 및 구현 역량**을 평가한다. 예선은 제안서 중심이고, 본선은 실제 동작 가능한 에이전트와 시연 산출물이 핵심이다.

## 3. 대회 방식

| 단계 | 공식 내용 요약 | 작업 해석 |
| --- | --- | --- |
| 온라인 예선 | 선택 분야 기반 에이전트 AI 신약개발 서비스를 기획하고 제안서 제출 | 지금 필요한 산출물은 10쪽 이내 제안서, 문제 정의, 에이전트 구조, 도구 활용 계획, 평가 계획이다. |
| 예선 평가 | 참가팀 상호평가 30% + 전문가 평가 70% | Peer Review용으로 10초 안에 이해되는 문제/효과 문장이 필요하고, 전문가용으로 근거/지표/아키텍처가 필요하다. |
| 본선 선발 | 10~15팀 선발 | 예선 제안서가 본선 구현 가능성을 보여야 한다. |
| 온라인 본선 | 약 4주간 실제 AI Agent 개발 및 결과물 구현 | 제안서 단계부터 MVP 범위를 4주 구현 가능하게 줄여야 한다. |
| 본선 제출 | 시연 영상, 연구/기술 보고서, 서비스 배포 링크, 발표자료 | 본선의 화면/로그/리포트 흐름을 예선 설계에 미리 반영해야 한다. |

## 4. 상세 주제 해석

| 분야 | 공식 의미 | 우리 해석 |
| --- | --- | --- |
| 분야 1: 자율형 가설 생성 및 검증 | 질병 타깃 발굴, 약물 재창출 가설 생성, 바이오마커 탐색 | 질병 입력을 받아 Open Targets, ChEMBL, 문헌 근거를 결합해 타깃-질병 가설을 만들고, 근거 강도와 검증 가능성을 점수화한다. |
| 분야 2: 도구 활용 기반의 분자 최적화 루프 | 분자 구조 최적화, 합성 경로 설계, 약물 유사성 개선 | 선택된 타깃의 seed molecule을 가져와 RDKit, ADMET 예측, 합성 가능성 평가를 반복하며 후보 구조를 개선한다. |
| 분야 4: 융합 분야 | 분야 1, 2, 3 또는 그 외 융합 | 우리는 분야 1+2 융합을 분야 4 성격으로 제출하되, 선택 분야는 "분야 1, 분야 2, 분야 4"를 모두 표시하는 전략을 검토한다. |

- 후속 지원: 우수 아이디어 사업화 지원 예정, 예: 국제 학술대회 참석/발표, 특허 출원

## 11. 제안 방향

이번 제안은 `질병-타깃 가설 생성`과 `분자 최적화`를 끊어서 설명하지 말고, 하나의 폐루프 시스템으로 설명해야 한다.

추천 메시지:

> 기존 AI 신약개발 도구는 타깃 가설과 분자 최적화가 분리되어 연구자가 반복 연결해야 한다. Hypothesis-to-Lead Agent는 질병 근거 그래프에서 검증 가능한 타깃 가설을 만들고, 해당 타깃의 seed molecule을 자동 수집/최적화/ADMET 평가하여 다시 가설 신뢰도를 갱신하는 에이전틱 AI다.
